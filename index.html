Features

Native Code Completion WIP

intellij-rust utilizes IntelliJ's built-in completion engine, completely independent from racer or any other 3rd-party utilities. A decent amount of completion intelligence is already implemented, but it's still at early stages of development.

Navigate Everywhere

  • Press Ctrl+B to jump to definition
  • Ctrl+n finds any struct or enum by name
  • Ctrl+Alt+Shift+N finds any symbol (functions, traits, modules, methods, fields) by name
  • Ctrl+u brings you to the parent of the current module

Code Formatter

Ctrl+Alt+l reformats current file using our custom, fully functional formatter based on IntelliJ'sformatting engine though rustfmt support is WIP

File Structure

Ctrl+F12 brings up an overview of the current file. It is usefull for navigation. Alt+7 presents the same information in the "project structure" style

Quick Documentation

Ctrl+q shows docs for the thing at caret

Live Templates

  • p, pd, ppd - print! value
  • a, ae - assert! / assert_eq!
  • tfn - test function boilerplate
  • tmod - test module boilerplate
  • f - field name and type
  • pf - pub field
  • loop, while, for surround templates

Intentions

  • Expand module: invoke this action inside foo.rs file to get foo/mod.rs.
  • Contract module: the opposite of expand module
  • Extract inline module: move mod foo { ... } to a separate file
  • Create module file: if you have unresolved mod foo; this intention will create foo.rs
  • Add derive clause: adds #[derive(..)] attribute to structs and enums
